**Function 1: `process_pdf_with_config(pdf_path, tesseract_config, dpi=200)`**

This function processes a single PDF with a custom Tesseract config string, bypassing the hard-coded config in `extract_id_with_rotation()`. It does NOT call `extract_id_with_rotation()` -- instead, it reimplements the rotation loop directly so it can inject the custom config without monkey-patching.

Implementation:
```python
def process_pdf_with_config(pdf_path, tesseract_config, dpi=200):
    """
    Process a single PDF with a specific Tesseract config string.

    Reimplements the rotation loop from extract_id_with_rotation() to allow
    config injection for benchmarking. Does NOT modify the main pipeline.

    Per D-01: Enables independent testing of each config variant.
    Per D-02: Reuses Phase 10 benchmark infrastructure patterns.

    Args:
        pdf_path: Path to PDF file
        tesseract_config: Full Tesseract config string (e.g., '--psm 6 --oem 1 ...')
        dpi: Rendering DPI (default 200, Phase 10 winner)

    Returns:
        List of result dicts with keys: filename, page, ids, rotation_detected, notes
    """
```

Logic:
1. Open PDF with `fitz.open(str(pdf_path))`
2. For each page: render at `dpi` with `page.get_pixmap(dpi=dpi, alpha=False)`, convert to PIL Image
3. For each page image, try rotations [90, 270, 0, 180] (same order as main pipeline)
4. For each rotation: call `pytesseract.image_to_string(rotated_image, config=tesseract_config)`
5. Apply `normalize_digits()` from precede_ocr (import it)
6. Apply `re.findall(r'\b\d{5}\b', normalized_text)` for ID matching
7. If matches found, call `select_all_valid_ids(matches)` from precede_ocr (import it)
8. If valid IDs, return result with ids, rotation, notes=''
9. If no direct match, try preprocessing fallback: call `preprocess_image(image)` from precede_ocr (import it), then retry all rotations
10. If preprocessing match, return with notes='preprocessed'
11. If no match, return with ids=[], notes='no_match'
12. Wrap entire function in try/except for PDF open errors, return error result
