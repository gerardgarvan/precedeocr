Create `precede_ocr.py` at the project root. This is the main pipeline script for Phase 1. It must implement the following functions and behavior:

If Poppler IS in PATH, set `POPPLER_PATH = None`.

**Function 1: `normalize_digits(text: str) -> str`**
- Purpose: Normalize common OCR digit confusion characters (per Pitfall 6 from research)
- Mapping dict: `{'O': '0', 'o': '0', 'I': '1', 'l': '1', '|': '1', 'S': '5', 's': '5', 'B': '8', 'b': '8', 'Z': '2'}`
- Iterate over mapping, replace each char in text
- Return normalized text

**Function 2: `select_most_likely_id(matches: list[str]) -> str | None`**
- Purpose: Per user decision D-03, when multiple 5-digit numbers found on a page, select the most likely Precede ID
- Filter out trivial/repeating patterns: `{'00000', '11111', '22222', '33333', '44444', '55555', '66666', '77777', '88888', '99999'}`
- If filtered list is non-empty, return first element
- If filtered list is empty but original matches is non-empty, return first match from original
- If matches is empty, return None

**Function 3: `extract_id_with_rotation(image: Image.Image) -> tuple[str | None, int | None]`**
- Purpose: Per user decision D-04, try OCR at all 4 rotations (0, 90, 180, 270) with early exit on first valid 5-digit match
- Loop over angles `[0, 90, 180, 270]`
- For angle 0: use image directly. For other angles: `image.rotate(angle, expand=True)`
- Tesseract config string: `'--psm 6 --oem 3 -c tessedit_char_whitelist=0123456789'`
  - PSM 6 (uniform text block) is a reasonable middle ground for full-page scans containing isolated IDs. If PSM 6 yields poor results, the user can adjust in later phases (Claude's discretion area).
  - OEM 3 uses the LSTM engine
  - Whitelist restricts output to digits only
- Run `pytesseract.image_to_string(rotated_image, config=config).strip()`
- Apply `normalize_digits()` to the OCR text
- Run `re.findall(r'\b\d{5}\b', normalized_text)` to find 5-digit matches with word boundaries (per Pitfall 7)
- If matches found: call `select_most_likely_id(matches)` and if result is not None, return `(result, angle)` — this is the early exit
- After all 4 rotations with no match: return `(None, None)`

**Function 4: `process_single_pdf(pdf_path: str) -> list[dict]`**
- Purpose: End-to-end pipeline for one PDF file (PIPE-01 single file, PIPE-02 DPI, PIPE-04 regex, PIPE-05 mapping)
- Create a temporary directory using `tempfile.mkdtemp(prefix='precede_ocr_')`
- Extract filename using `Path(pdf_path).name`
- Convert PDF to images using `convert_from_path()` with these exact parameters:
  ```python
  convert_from_path(
      pdf_path,
      dpi=300,                    # PIPE-02: 300+ DPI for reliable digit recognition
      output_folder=temp_dir,     # Pitfall 1: disk-backed, not RAM
      paths_only=True,            # Pitfall 1: returns file paths, prevents OOM
      fmt='png',                  # Lossless format preserves OCR quality
      poppler_path=POPPLER_PATH   # None if in PATH, else explicit path
  )
  ```
- Initialize empty `results` list
- Loop over `enumerate(image_paths, start=1)` to get `(page_num, image_path)`:
  - Open image with context manager: `with Image.open(image_path) as img:` (per Pitfall 5)
  - Call `extract_id_with_rotation(img)` to get `(id_found, rotation)`
  - Append dict to results per D-06 (row for EVERY page, even no-match) and D-07 (column order):
    ```python
    {
        'filename': filename,
        'page': page_num,
        'id': id_found,           # None if no match — will be blank in CSV
        'rotation_detected': rotation if id_found else None
    }
    ```
- After processing all pages, clean up temp directory: `shutil.rmtree(temp_dir, ignore_errors=True)`
- Return results list

**Function 5: `write_results_csv(results: list[dict], output_path: str) -> None`**
- Purpose: Write results to CSV per OUT-01 and D-07
- Create output directory if it doesn't exist: `Path(output_path).parent.mkdir(parents=True, exist_ok=True)`
- Create DataFrame: `pd.DataFrame(results)`
- Enforce column order per D-07: `df = df[['filename', 'page', 'id', 'rotation_detected']]`
- Write CSV: `df.to_csv(output_path, index=False)`
- Print summary to stdout:
  ```
  Results written to {output_path}
  Total pages scanned: {len(df)}
  IDs found: {count of non-null ids}
  Pages with no ID: {count of null ids}
  ```

**Main block (`if __name__ == '__main__':`):**
- Accept PDF path as command-line argument: `sys.argv[1]` if provided, else print usage and exit
- Validate the PDF file exists using `Path(pdf_path).is_file()` — if not, print error and exit with code 1
- Also accept optional second argument for output CSV path, defaulting to `output/results.csv`
- Call `process_single_pdf(pdf_path)` to get results
- Call `write_results_csv(results, output_path)` to write CSV
- Print "Done." at the end
